# 网络设备巡检工具 v2.0

> 本项目仅供学习和交流使用，不得用于商业发布或未授权的运维操作。

## 项目概述

这是一个现代化的网络设备巡检工具，基于Python和Tkinter构建，使用Netmiko库连接网络设备，支持SSH和Telnet协议。该工具提供直观的图形界面，可批量对网络设备执行巡检任务，并生成详细的结果报告。

## 功能特点

- **现代化UI设计**：采用卡片式布局和浅色主题，提供美观的用户体验
- **多协议支持**：支持SSH和Telnet协议连接设备
- **批量巡检**：可同时对多个设备执行巡检任务
- **主题切换**：支持浅色主题和高对比度模式
- **实时日志**：显示巡检过程中的详细日志信息
- **多线程处理**：支持并发执行巡检任务，提高效率
- **设备管理**：支持设备列表和设备类型的CSV/文本文件配置
- **搜索过滤**：支持实时搜索设备列表
- **结果导出**：可将巡检结果导出为Excel文件

## 技术架构

### 核心模块

1. **主界面模块** (`ModernNetworkInspectionUI`)
   - 负责创建和管理GUI界面
   - 包含设备列表、工具栏、日志面板等组件
   - 实现主题管理和UI事件处理

2. **连接模块** (`connect_and_execute`)
   - 使用Netmiko库连接网络设备
   - 支持SSH和Telnet协议
   - 自动处理认证和连接异常

3. **巡检引擎模块** (`inspection_worker`)
   - 执行具体的巡检任务
   - 管理多线程巡检进程
   - 处理设备命令执行和输出解析

4. **配置管理模块**
   - 加载和管理设备列表
   - 支持设备类型和命令配置文件

### UI组件

1. **ModernFrame** - 现代化框架容器
2. **ModernButton** - 现代化按钮
3. **ModernEntry** - 带占位符的输入框
4. **DetailedProgressbar** - 带标签的详细进度条
5. **DeviceListWidget** - 设备列表组件

### 主题管理系统

- **ThemeManager** - 管理浅色和高对比度主题的颜色方案
- 支持动态切换主题，无需重启应用

## 执行流程

### 启动流程

1. 初始化`ModernNetworkInspectionUI`实例
2. 加载配置文件（设备列表、设备类型、命令配置）
3. 创建主窗口和UI组件
4. 初始化主题管理器
5. 设置事件监听器和快捷键
6. 显示主窗口

### 巡检流程

1. 用户点击"开始巡检"按钮
2. 检查选中的设备数量，若为0则提示用户
3. 获取并发数设置和编码配置
4. 禁用开始按钮，启用停止按钮，更新状态指示器
5. 启动巡检工作线程，执行以下步骤：
   - 为每个选中的设备创建线程
   - 连接到设备
   - 执行预设命令
   - 解析命令输出
   - 记录巡检结果
   - 更新进度条和状态
6. 所有设备巡检完成后，重新启用开始按钮，禁用停止按钮
7. 生成巡检报告

### 线程管理

- 主线程负责UI渲染和事件处理
- 巡检任务在独立的工作线程中执行
- 使用ThreadPoolExecutor管理线程池
- 通过队列在线程间传递消息和数据
- 使用线程锁保护共享资源

## 使用说明

### 启动程序

```bash
python network_inspection_v2.0_modern.py
```

### 基本操作

1. **加载设备列表**：菜单 > 设置 > 加载设备列表
2. **加载设备类型**：菜单 > 设置 > 加载设备类型
3. **配置命令文件**：菜单 > 设置 > 配置设备命令
4. **开始巡检**：点击"开始巡检"按钮
5. **停止巡检**：点击"停止巡检"按钮
6. **查看结果**：在设备列表中查看各设备状态和结果
7. **导出报告**：点击"导出结果"按钮
8. **打开日志目录**：点击"日志目录"按钮

### 主题切换

通过菜单 > 设置 > 高对比度模式来切换浅色/高对比度主题。
